# Autobuy

Autobuy is a powerful feature that allows you to automatically buy items with the click of a button. It is available for the Classic and Pro subscriptions.

## Setup

In order to use the autobuy feature, you will neede to provide your Vinted token. You can find it by following the steps in the following video:

<ReactPlayer
  playing
  controls
  url="https://cdn.discordapp.com/attachments/1108476655388872846/1109412710963220550/vinted-chrome-en.mp4"
/>

Once you have obtained your token, you can use the `/token` command to set it:

```
/token <token>
```

## Usage

Once you have set your token, every message will inclue a button that allows you to buy the item. Simply click on the button to buy the item.

![Autobuy button](/img/autobuy-button.png)

:::danger Warning
Autobuy is a powerful feature that will use the money in your vinted balance. Please use it with caution.
:::

:::tip Looking for more?
Check out the [Pro subscription](https://vintedbot.com/pricing) to get access to a custom autobuy that also works without money in your balance.
:::
